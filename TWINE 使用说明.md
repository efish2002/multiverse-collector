## 🆕 v0.3 新功能说明

### 💾 存档系统

#### 如何使用
1. 点击顶部菜单栏的 **"💾 存档管理"** 按钮
2. 选择一个存档位（1/2/3）
3. 点击 **"💾 保存当前进度"** 保存游戏
4. 下次游戏时，点击 **"📂 读取选中存档"** 恢复进度

#### 存档信息
每个存档显示：
- 当前所在宇宙
- 已收集概念数量
- 当前得分
- 存档时间

#### 注意事项
- 存档保存在浏览器 LocalStorage 中
- 清除浏览器数据会删除存档
- 不同浏览器/设备存档不共享
- 建议定期保存重要进度

### 📦 概念图鉴

#### 如何查看
1. 点击顶部菜单栏的 **"📦 概念图鉴"** 按钮
2. 浏览所有 85+ 个概念卡片

#### 图鉴功能
- **已收集概念**：显示完整信息（描述、科学原理、游戏效果）
- **未收集概念**：显示锁定状态（🔒），名称和描述隐藏
- **稀有度标识**：普通（灰色）、稀有（绿色）、史诗（紫色）、传说（金色）
- **收集进度**：显示已收集/总数

### 🏆 成就系统

#### 如何查看
1. 点击顶部菜单栏的 **"🏆 成就系统"** 按钮
2. 查看所有成就的完成状态

#### 成就列表
| 成就名称 | 解锁条件 |
|----------|----------|
| 🏆 概念新手 | 收集第一个概念 |
| 🌌 宇宙跳跃者 | 穿越 5 个平行宇宙 |
| 📦 收集达人 | 收集 10 个概念 |
| 🎯 收集大师 | 收集 50 个概念 |
| 👑 传奇收集者 | 收集 100 个概念 |
| 🗺️ 探索者 | 访问所有场景 |
| 🔀 命运编织者 | 做出 50 个选择 |
| ⭐ 高分玩家 | 得分达到 1000 |

#### 成就通知
- 解锁成就时会显示弹窗通知
- 成就自动检测，无需手动领取

### 🔄 重新开始
- 点击 **"🔄 重新开始"** 按钮
- 确认后即可重置游戏进度
- **注意**：未保存的进度将丢失！

## 🔧 自定义与扩展

### 添加新场景

在 `scenes` 对象中添加：

```javascript
"new_scene": {
    title: "场景标题",
    text: "场景描述文本",
    choices: [
        { text: "选项 1", next: "next_scene_id" },
        { text: "选项 2", next: "another_scene", concept: "概念名" }
    ],
    onEnter: () => collectConcept("自动获得的概念")
}
```

### 添加新概念

在 `collectConcept()` 函数中会自动处理，无需额外代码。

### 修改样式

编辑 `<style>` 部分：
- 修改 `background` 改变主题色
- 修改 `.choice-btn:hover` 改变按钮效果
- 修改 `.concept-collected` 改变获得概念动画
