## 🌍 一、世界观设定

### 核心概念：**"多元宇宙概念收集者"**

```
背景故事：
公元 2147 年，人类发现宇宙是由无数"概念"构成的信息结构。
某些特殊个体（称为"收集者"）能够穿越平行宇宙，
收集不同宇宙的科幻概念，用于修复濒临崩溃的主宇宙。

玩家角色：
- 身份：第 9 代概念收集者
- 使命：收集 1000 个科幻概念，修复主宇宙
- 能力：宇宙跳跃、概念具现化、时间暂停（有限）
```

|------|----------|----------|------|
| **收集大师** | 概念新手 | 收集 10 个概念 | 新宇宙解锁 |
| | 概念猎人 | 收集 50 个概念 | 跳跃能力升级 |
| | 概念大师 🌟 | 收集 100 个概念 | 特殊称号 |

| | 时间旅行者 | 探索时间旅行宇宙 | 时间暂停 +5 秒 |


### 2.2 🎯 任务链系统

#### 主线任务（修复主宇宙）

```
第一章：觉醒
├─ 任务 1-1：初次跳跃（教程）
├─ 任务 1-2：收集 5 个基础概念
└─ 任务 1-3：修复第一个宇宙碎片

第二章：成长
├─ 任务 2-1：探索 3 个不同宇宙
├─ 任务 2-2：收集 20 个概念
└─ 任务 2-3：第一次重大抉择（分支点）

第三章：挑战
├─ 任务 3-1：对抗"概念吞噬者"
├─ 任务 3-2：收集 50 个概念
└─ 任务 3-3：牺牲还是拯救？（关键抉择）

第四章：终局
├─ 任务 4-1：集齐 100 个概念
├─ 任务 4-2：最终抉择（多结局）
└─ 任务 4-3：新宇宙诞生（结局）
```

#### 支线任务（各宇宙特色）

```
赛博朋克宇宙支线：
- "神经网络的秘密" - 潜入企业数据库
- "AI 的觉醒" - 帮助 AI 获得自由
- "义体升级计划" - 收集升级部件

太空歌剧宇宙支线：
- "星际外交官" - 调解外星种族冲突
- "失落星球考古" - 发掘古代文明
- "超光速竞赛" - 飞船竞速挑战

时间旅行宇宙支线：
- "祖父悖论" - 修复时间线错误
- "历史观察者" - 见证关键历史事件
- "因果律挑战" - 改变过去的影响
```

#### 探索机制

- **动态事件**: 随机触发特殊事件（限时）
- **环境叙事**: 通过场景细节讲述背景故事

### 2.4 🔄 选择与后果系统

#### 关键决策点

```
决策点 1（第二章末尾）：
选择 A：拯救赛博朋克宇宙的 AI
  → AI 成为盟友，解锁黑客技能
  → 企业成为敌人，后续任务难度↑
  → 结局倾向：技术乌托邦

选择 B：摧毁失控 AI
  → 企业成为盟友，获得资金支持
  → AI 幸存者成为敌人
  → 结局倾向：人类至上

决策点 2（第三章末尾）：
选择 A：牺牲一个宇宙拯救主宇宙
  → 主宇宙修复度 +50%
  → 失去一个宇宙（永久关闭）
  → 结局倾向：功利主义

选择 B：寻找两全其美的方法
  → 需要额外收集 20 个概念
  → 两个宇宙都保留
  → 结局倾向：理想主义
```

#### 后果追踪系统

```
影响维度：
1. 盟友/敌人关系变化
2. 可用技能/道具变化
3. 可访问区域变化
4. 结局分支解锁
5. NPC 态度变化

可视化：
- "因果网"界面显示所有决策的影响链
- 时间线视图展示不同选择导致的分支
```

## 📖 三、叙事框架

### 叙事结构

```
非线性叙事 + 碎片化叙事

主线：清晰的任务链引导
支线：自由探索发现
背景：通过概念卡片、环境细节、NPC 对话碎片化呈现

玩家体验：
1. 初期：线性引导（教程）
2. 中期：半开放（主线 + 支线）
3. 后期：完全开放（自由探索 + 多结局）
```

### 角色设定

```
主要 NPC：
1. 导师 - "第 8 代收集者"（已失踪）
   - 功能：教程引导、 hints 提供
   - 秘密：为何失踪？

2. 对手 - "概念掠夺者"
   - 功能：竞争、冲突
   - 动机：收集概念用于个人目的

3. 盟友 - 各宇宙原住民
   - 功能：任务发布、交易、情感连接
   - 特色：每个宇宙独特 NPC
```
