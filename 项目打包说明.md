## 🚀 使用方式

### 方式 1：直接打开（推荐）
```bash
# 解压压缩包
tar -xzvf multiverse-concept-collector-v0.21.tar.gz
cd game-scifi-hybrid/

# 直接在浏览器打开主文件
# 文件路径：twine_version_v18_expanded.html
```

### 方式 2：本地 Web 服务器
```bash
# 进入项目目录
cd game-scifi-hybrid/

# 启动 Python 简单 HTTP 服务器
python3 -m http.server 8000

# 浏览器访问
http://localhost:8000/twine_version_v18_expanded.html
```

### 方式 3：部署到 Web 服务器
```bash
# 使用部署脚本（需配置）
./deploy_v0.13.sh

# 或手动部署到 Nginx/Apache
cp twine_version_v18_expanded.html /var/www/html/game.html
```
