### 1. 得分计算修复 ⭐⭐⭐⭐⭐

**问题：** 收集概念时只加 10 分，没有按稀有度计算；宇宙探索没有奖励分数。

**修复内容：**

  - 史诗 (epic): 30 分
  - 传说 (legendary): 50 分
- 在每个宇宙的 `onEnter` 回调中添加 100 分探索奖励
- 确保 `updateStatus()` 在收集概念后立即调用

**代码位置：**
```javascript

const scoreGain = rarityScores[rarity] || 10;

// 每个宇宙的 onEnter

```

### 5. 滚动行为修复 ⭐⭐⭐⭐

**问题：** 每次点击选择支都跳转到页面顶部。

**修复内容：**
- 修改 `renderScene()` 函数
- 将 `window.scrollTo({ top: 0, behavior: 'smooth' })` 改为 `container.scrollIntoView({ behavior: 'smooth', block: 'start' })`
- 现在只滚动到游戏内容区顶部，而不是整个页面顶部

**代码位置：**
```javascript
// renderScene 函数末尾
const container = document.getElementById('game-container');
if (container) {
    container.scrollIntoView({ behavior: 'smooth', block: 'start' });
}
```
