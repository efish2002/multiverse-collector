## 🛠️ 生成工具推荐

### 在线工具
1. **Midjourney** (https://www.midjourney.com/) - 最佳质量，需订阅
2. **DALL-E 3** (https://chat.openai.com/) - 通过ChatGPT使用
3. **Bing Image Creator** (https://www.bing.com/create) - 免费，基于DALL-E
4. **Leonardo.ai** (https://leonardo.ai/) - 免费额度充足
5. **Playground AI** (https://playgroundai.com/) - 免费版可用

### 本地工具
1. **Stable Diffusion** + **ControlNet** - 完全免费，需显卡
2. **ComfyUI** - 节点式工作流，灵活度高

### 批量生成建议
- 使用相同的种子值保持风格一致
- 添加 `"--style raw"` (Midjourney) 或 `"photorealistic, cinematic"` 保持真实感
- 建议尺寸：16:9 或 1920x1080 用于场景展示
