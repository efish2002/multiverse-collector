## 📝 技术总结

### 使用的技术
- **HTML5** - 页面结构
- **CSS3** - 样式和动画（渐变、过渡、关键帧）
- **JavaScript ES6+** - 游戏逻辑
- **LocalStorage API** - 存档存储
- **DOM 操作** - 动态界面更新

### 关键函数
```javascript
// 存档核心
saveGame()       // 保存游戏
loadGame()       // 读取游戏
deleteSave()     // 删除存档
updateSaveSlots() // 更新存档显示

// 图鉴系统
renderConceptGallery() // 渲染概念图鉴
showConcepts()         // 显示图鉴界面

// 成就系统
renderAchievements()   // 渲染成就列表
checkAchievements()    // 检测成就解锁
showAchievementNotification() // 显示成就通知

// 界面管理
toggleSaveSystem()     // 切换存档界面
showConcepts()         // 显示图鉴
showAchievements()     // 显示成就
resetGame()            // 重置游戏
```
