## 💡 开发建议

### 代码风格
```javascript
// ✅ 好的命名
function collectConcept(conceptId) { }
const gameData = { concepts: [], score: 0 };

// ❌ 避免
function a() { }
const d = {};
```

### 场景设计
- 保持每个场景 300-500 字
- 每个场景 3-5 个选择
- 确保所有选择都有后续（避免死胡同）
- 添加情感元素（让玩家关心角色和故事）

### 概念设计
- 描述要有趣且科学准确
- 游戏效果要平衡（不过强也不过弱）
- 与世界观关联（不是孤立的概念）

### 常见问题
**Q: 如何添加新场景？**  
A: 在 `scenes` 对象中添加新条目，参考现有场景格式。

**Q: 如何实现概念融合？**  
A: 添加 `fuseConcepts(concept1, concept2)` 函数，检查配方并返回新概念。

**Q: 如何添加音效？**  
A: 使用 HTML5 Audio API，在相应事件触发时播放。
